# Phase 5 竞品差异一页纸

## 1. 我们解决的问题

市场洞察工具常见痛点：

- 只给结论，不给过程；
- 能回答但不可复核；
- 现场演示一断网就崩；
- 结果难以结构化交付给业务团队。

WeaveAI 2.0 的定位是：**可辩论、可追溯、可导出的市场决策助手**。

## 2. 横向对比

| 维度 | 普通单 Agent 问答 | 常规报告生成工具 | WeaveAI 2.0 |
| --- | --- | --- | --- |
| 推理结构 | 单线程 | 模板化流水 | Supervisor-Worker + 多 Agent 辩论 |
| 过程可见性 | 低 | 中 | 高（进度、辩论、降级全可见） |
| 证据追溯 | 弱 | 中 | 强（Evidence Pack + 覆盖率） |
| 稳定性韧性 | 弱 | 中 | 强（演示保护模式 + 90 秒恢复） |
| 成本观测 | 低 | 中 | 高（tool_metrics + demo_metrics） |
| 交付形态 | 文本片段 | 报告文件 | 路演资产包 zip（报告+证据+指标） |

## 3. 差异化能力（评委可见）

### A. 评委模式

- 预置 3 套场景（60 秒极速 / 3 分钟标准 / 深度演示）。
- 现场无需调参数，支持一键启动。

### B. 关键指标看板

- 耗时、稳定性评分、证据覆盖率、降级次数。
- 把“效果”从主观感受变成结构化指标。

### C. 演示保护模式

- 流中断自动回补状态。
- 恢复中/已恢复/超时状态可视化。

### D. 一键导出路演包

- 输出可复审、可归档、可传阅的资产集合。

## 4. 进入业务的最短路径

1) 评委模式跑通标准场景；
2) 用关键指标看板量化稳定性和可信度；
3) 导出路演包沉淀到业务评审流程。

## 5. 结论（对外一句话）

> WeaveAI 2.0 的优势不在“会不会写报告”，而在“是否能稳定地产出可追溯、可复核、可交付的决策资产”。
